## 🚀 What is this?
Hadistpedia isn't your average, clunky religious app from 2012. It's a next-gen, aesthetically pleasing hub for exploring authentic Hadiths, daily reflections, and Islamic wisdom. Built from the ground up with **Flutter** and powered by a totally based **Domain-Driven Design (DDD)** architecture, this app is literally built different. 

Whether you're looking for your *Daily Vibe* or doing a deep dive into Sahih Bukhari, Hadistpedia got you covered. No cap.

## 💅 The Aesthetics
We took "UI/UX" and leveled it up:
- **Glassmorphism**: Frosted glass UI that effortlessly passes the vibe check.
- **Cosmic Glows**: Dark mode supremacy with subtle pink and purple nebula glows. Literally eye candy.
- **Buttery Smooth**: Edge-to-edge layout with bouncing scroll physics. 0 lag. 100% drip.

## 🛠 Tech Stack (The Nerd Stuff)
We aren't just making it look pretty; the code underneath is highly structured and professional. 
- **Framework**: Flutter 💙
- **Architecture**: Domain-Driven Design (DDD) - *because spaghetti code is a massive red flag.*
  - `/data`: Where the APIs, data sources, and JSON parsing live.
  - `/domain`: The core business logic and entities (clean AF).
  - `/presentation`: Where the UI magic happens.
- **Data Handling**: Ready for complex, multi-lingual JSON structures with robust models.

## 🔥 Key Features
- **Daily Vibe**: Get hit with a randomly curated, highly authentic daily quote that lowkey speaks exactly to your soul.
- **Deep Database**: Structured to handle everything—Arabic text, multi-language translations, detailed narrators, and strict scholar authenticity grades.
- **Smart Filtering**: Seamless category chips (Hadith, Reflections, Audio) so you can find exactly what you need without the hassle.

## 💻 How to spin this up
Don't gatekeep, here's how to run it locally:

```bash
# Clone the repository
git clone https://github.com/your-username/hadistpedia.git

# Pull up to the directory
cd hadistpedia

# Get the dependencies
flutter pub get

# Run it on your simulator or device
flutter run
```
